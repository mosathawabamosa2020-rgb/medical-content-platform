# Risk Register

| Risk | Mitigation |
| --- | --- |
| Over-reliance on AI without source governance | Scientific verification gate mandatory |
| Weak reference quality | Quality scoring at ingest, minimum threshold configurable |
| No version tracking | KnowledgeRevision table captures edits and lineage |
| Premature service expansion | Modular monolith boundaries and deferred roadmap gates |
| Offline AI unrealism in Phase 1 | OpenAI in Phase 1, self-hosted embedding migration planned in Phase 2 |
