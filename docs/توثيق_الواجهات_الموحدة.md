# توثيق الواجهات الموحدة

## 1. نظرة عامة

تم إنشاء نظام واجهات موحد يتضمن:
- شريط تنقل رئيسي موحد
- تصميم عربي متكامل
- تخطيط متجاوب
- جميع الصفحات تستخدم نفس التخطيط

## 2. المكون الرئيسي: Layout

### 2.1 الملف
- المسار: `components/Layout.tsx`
- الوظيفة: توفير تخطيط موحد لجميع الصفحات

### 2.2 المكونات
1. **شريط التنقل الرئيسي**
   - شعار المنصة
   - روابط التنقل الرئيسية
   - حالة المستخدم
   - زر تسجيل الدخول/الخروج

2. **المحتوى الرئيسي**
   - حاوية المحتوى
   - عرض محتوى الصفحة

3. **التذييل**
   - معلومات المنصة
   - روابط سريعة
   - معلومات الدعم الفني

## 3. الصفحات الموحدة

### 3.1 الصفحة الرئيسية
- المسار: `pages/index.tsx`
- الميزات:
  - استخدام التخطيط الموحد
  - عرض واضح للبحث
  - عرض المراجع المميزة
  - عرض المراجع الحديثة

### 3.2 صفحة الأجهزة
- المسار: `pages/devices/index.tsx`
- الميزات:
  - استخدام التخطيط الموحد
  - عرض جميع الأجهزة
  - عرض المراجع المتصلة
  - إحصائيات واضحة

### 3.3 صفحة المكتبة
- المسار: `pages/library/index.tsx`
- الميزات:
  - استخدام التخطيط الموحد
  - عرض جميع المراجع
  - تصفية المراجع
  - إحصائيات واضحة

### 3.4 صفحة إنشاء المحتوى
- المسار: `pages/create/index.tsx`
- الميزات:
  - استخدام التخطيط الموحد
  - اختيار الأجهزة
  - اختيار المراجع
  - إعدادات المحتوى

### 3.5 صفحة تفاصيل المرجع
- المسار: `pages/reference/[id].tsx`
- الميزات:
  - استخدام التخطيط الموحد
  - عرض تفاصيل المرجع
  - عرض الأقسام
  - أزرار الإجراءات

## 4. التصميم

### 4.1 الألوان
- الأزرق: الأزرار الرئيسية والروابط
- الأخضر: حالة النجاح والنشاط
- الأصفر: حالة الانتظار
- الأحمر: حالة الخطأ والرفض
- الرمادي: النصوص والأوصاف

### 4.2 التخطيط
- استخدام Tailwind CSS للتنسيق
- تصميم متجاوب مع جميع الأحجام
- توزيع واضح للمحتوى
- مسافات مناسبة بين العناصر

### 4.3 التجاوب
- متجاوب مع الهواتف
- متجاوب مع الأجهزة اللوحية
- متجاوب مع جميع الأحجام
- توزيع واضح للمحتوى

## 6. التحقق

### 6.1 التحقق من الصفحة الرئيسية
```bash
# التحقق من البحث في قاعدة البيانات
curl -X POST http://localhost:3000/api/references/query \
  -H "Content-Type: application/json" \
  -d '{"query": "PCR", "topK": 8, "page": 1}'

# التحقق من البحث في PubMed
curl "http://localhost:3000/api/search/pubmed?term=diabetes"
```

### 6.2 التحقق من صفحة الأجهزة
```bash
# التحقق من الأجهزة
curl http://localhost:3000/api/devices
```

### 6.3 التحقق من صفحة المكتبة
```bash
# التحقق من المراجع
curl http://localhost:3000/api/references/library?featured=true&limit=6
```
